> A Rust wrapper around the [Safaricom API](https://developer.safaricom.co.ke/docs?shell#introduction) for accessing M-Pesa services.

## Build

```sh
cargo build
```

## Usage

```sh
cargo run
```

## Run tests

```sh
cargo test
```
